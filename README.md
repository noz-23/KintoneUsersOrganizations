# KintoneUsersOrganizations

## ユーザー組織展開プラグイン

## 1.概要

ユーザー選択後、所属する全組織と優先組織を展開するプラグインです。
選択できるユーザー数は制限を設けることができ、全組織と優先組織は任意に設定できます。

## 2.今後

現行はユーザー選択(変更)時に、変更する仕様となります。
展開ボタンなどの押下後などの任意のタイミングでも展開できるようにもできる用にしたいと思います。
